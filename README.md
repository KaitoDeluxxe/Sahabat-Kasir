Sahabat Kasir Google Appscript Setup

1. Buat Google Spread Sheet Baru
2. Pilih menu Extensions/Ekstensi, lalu pilih APP SCRIPT
3. Copy isi dari APP_SCRIPT.md
4. Pilih Deploy>Create Deployment> Select Type, bisa pilih Web App
5. Nama dan Versi bisa disesuaikan dengan keinginan, isi Who has Access sebagai Anyone atau Siapa saja.
6. Authorize Access lalu pilih akun google anda.
7. pilih advance dan klik Go to XX project (unsafe), lalu lanjutkan/continue.
8. copy web app link yang muncul, lalu paste pada Tab Financial Settings di aplikasi
9. lakukan Backup data untuk membuat sheet yang diperlukan secara otomatis di Spreadsheet Anda.
